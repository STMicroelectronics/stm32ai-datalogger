Detailed Description

Wrapper class that wraps low level functions from AILoggingBackend to 'Python' level functions providing the C library services from a more 'Pythonic' point of view.

This class also wraps send/receive functions and buffers directly into itself to avoid manually declaring them in Python. User can use this middle-level API to suits their needs.

Parameters
send_functionthe send function that will be used to send the data. Function need to have 1 parameter that is the data to send
receive_functionfunction that will be used to receive data. This function take a size to read as parameter and returns the data received
receive_fptr_read_lengththe default size to read data from receive function
send_buffer_sizethe default size of send_buffer (if data is bigger than buffer_size, the packet will be send directly, not copied internally)
receive_buffer_max_sizesize of the reception buffer
raw_buffer_sizesize of the reception buffer for raw type packets
